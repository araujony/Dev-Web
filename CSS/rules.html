Em CSS, usamos a Font-family para específicar a fonte de um texto.

A Font-style propriedade é usada principalmente para especificar texto em itálico.
Esa propriedade tem três valores:

  • Normal - o texto é mostrado normalmente
  • Itálico - o texto é mostrado em itálico
  • Oblíquo - o text0 está "inclinado" (Oblíquo é muito semelhante ao itálico mas menos suportado)

A Font-size define tamanho do texto. Ser capaz de gerenciar o tamanho do texto
é importante no web design. No entanto,
você não deve usar ajustes de tamanho
de fonte para fazer com que os parágrafos
pareçam títulos, ou os títu
los se
parecem com parágrafos.
font-size: 40px;
font-size: 40rem;

A propriedade CSS font

Para encurtar o código, também é possível especificar todas as propriedades de fonte individuais em uma propriedade. A font é uma propriedade abreviada para:
  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family
Nota:Os valores font-size e font-family são obrigatórios. Se um dos outros valores estiver ausente, seu valor padrão será usado.
